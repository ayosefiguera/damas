# Damas
Damas es una simple implementación del juego de mesa realizado en python y 
haciendo uso de la librería Pygame.

## Requerimientos

* Python 3.8 o superior
* Entorno. python3 -m venv env
* Pygame pip install pygame==2.0.0.dev10

## Setup

pipenv install --dev


## Development

### Run tests

pipenv run test

### Run linter

pipenv run lint
